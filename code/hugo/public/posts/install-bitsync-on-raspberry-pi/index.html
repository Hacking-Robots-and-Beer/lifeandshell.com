Install Bitsync on Raspberry Pi

2013-09-13
1 minute read
Code , Life , Linux
Code , Life , Linux

So today im using dropbox to sync all my stuff between devices. But now there are so much there so my free space is almost full. So now its time for me to move to bitsync an then sync all my devices.

Install bitsync

Go to folder /opt

cd /opt

Download bitsync

wget "http://btsync.s3-website-us-east-1.amazonaws.com/btsync_arm.tar.gz"

unpack it

chmod 700 btsync_arm.tar.gz
tarr zxvf btsync_arm.tar.gz

Start it

cd bitsync
./bitsync

go to the webbpage

http:// IP-ADDRESS:8888

And set  what folder on the raspberry you want to sync.

1. Set up server

on the raspberry choose one server that you would like to sync.
And generat an key for that folder.

2. Set up client to sync to server.

Go to the webbpage http://labs.bittorrent.com/experiments/sync/other-download.html
and down load the client that you use.

– Then start bitsync
– Choose en folder on the client that you want to sync to the server
– PAST IN THE KEY FROM THE SERVER TO THAT FOLDER

 

DONE see you client when its syncing folder for you
